# Brave Origin Patcher Utilities
![Alt Text](SmallLogo.png)

## UNDER MAINTENANCE


This repo contains two simple utilities, one for MacOS, one for Windows 10/11 that allow you to get Brave Origin for free, telling Brave Origin that you purchased the product, even if you didn't!

### Features: 
* No prerequsite software or manual file editing.
* Easy apply, easy revert with simple GUI.
* Works with Brave Origin, Brave Origin Beta and Brave Origin Nightly.
* Windows **and** MacOS versions.


### Usage:

1. Download the latest release from the releases section relevent to your OS.
2. Download and install Brave Origin. After installing, close the purchase dialog.
3. OS-specific instructions:
* **Windows:** Double-click on the executable, select the version(s) you want patched, and it'll patch Brave Origin.
* **MacOS:**
  * Open and expand the ZIP Archive.
  * Double click on the Brave Origin Unlocker.
    * If it says that "Apple could not verify that is app is free from malware", then disable Gatekeeper.
    * To disable Gatekeeper, type in terminal `xattr -dr com.apple.quarantine /path/to/App.app` with your path to the app.
  * 
4. That's it! Hopefully... Please report any issues!
